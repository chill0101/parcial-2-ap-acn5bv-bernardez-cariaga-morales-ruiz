# Track IT

Plataforma de gestión centralizada de activos IT para PyMEs.

## Cómo correr
```bash
cd web
npm install
npm run dev    # http://localhost:5173
```

## Scripts
- `npm run dev` — servidor de desarrollo
- `npm run build` — build producción
- `npm run preview` — previsualizar build
